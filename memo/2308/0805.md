1. 프론트에서 백으로 요청을 보낸다.

```jsx
<a href={"<백앤드주소>"}> 카카오로그인 </a>
```

2. 백은 프론트를 카카오 로그인 페이지로 리다이렉트 시킨다.

controller에 이런식으로 함수를 짜주면 된다.

```jsx
        toKaKaoLoginPage(@Res res) {
                res.status(200).redirect("<url here>")
        }
```

이 리다이렉트 주소에 query와 함께 카카오 로그인 페이지 url를 넣어주면 된다.
